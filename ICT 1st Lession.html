දත්ත සහ තොරතුරු Data vs. Information

දත්ත සහ තොරතුරු බොහෝ විට එකිනෙකට හුවමාරු වන වචන වුවද, ඒවාට වෙනස් අර්ථයන් ඇත. දත්ත යනු අමු, සකස් නොකළ කරුණු වන අතර, තොරතුරු යනු අර්ථවත් සහ තීරණ ගැනීමට ප්‍රයෝජනවත් වන ලෙස සකස් කරන ලද දත්ත වේ. මෙම කොටස එම වෙනස්කම් සහ තොරතුරුවල වටිනාකම ගවේෂණය කරයි. While often used interchangeably, data and information have distinct meanings. Data is raw, unprocessed fact, while information is data that has been processed to be meaningful and useful for decision-making. This section explores those differences and the value of information.

දත්ත (Data)Data

නිශ්චිත නිගමන වලට එළඹිය නොහැකි අංක, වචන, සලකුණු, රූප.Numbers, words, symbols, images that cannot be drawn to definite conclusions.

දත්ත වර්ගTypes of Data

  • ප්‍රමාණාත්මක (Quantitative):Quantitative: සංඛ්‍යාත්මකව ඉදිරිපත් කළ හැකි. (උදා: උස, වයස)Can be presented numerically. (e.g., height, age)
  • ගුණාත්මක (Qualitative):Qualitative: සංඛ්‍යාත්මකව ඉදිරිපත් කළ නොහැකි. (උදා: පැහැය, හැඩය)Cannot be presented numerically. (e.g., color, shape)

තොරතුරු (Information)Information

නිශ්චිත නිගමනයකට හෝ තීන්දු වලට එළඹිය හැකි ලෙස සකස් කරන ලද දත්ත.Data prepared in such a way that specific conclusions or decisions can be reached.

තොරතුරක ලක්ෂණCharacteristics of Information

  • අර්ථවත් විය යුතුයMust be meaningful
  • පෙර දැනුම යාවත්කාලීන කරයිUpdates prior knowledge
  • අන්තර් සන්නිවේදන මාධ්‍යයකිIs a media of intercommunication
  • කාලීන වේIs timely
  • තීරණ ගැනීමට උපකාරි වේHelps in decision making

ගුණාත්මක තොරතුරක ලක්ෂණCharacteristics of Quality Information

  • ආදාලත්වය (Relevance)Relevance
  • සම්පුර්ණ බව (Completeness)Completeness
  • නිවැරදි බව (Correctness)Correctness
  • කාලීන බව (Timeliness)Timeliness
  • පැහැදිලි බව (Clarity)Clarity
  • විශ්වාසනීයත්වය (Reliability)Reliability

තොරතුරක ස්වර්ණමය නීතිය Golden Rule of Information

තොරතුරක වටිනාකම උපරිම මට්ටමක පවතින්නේ එය නිර්මාණය වු අවස්ථාවේදීය. කාලය ගත වීමත් සමග තොරතුරක වටිනාකම අඩු වී යයි. මෙම ප්‍රස්ථාරය එම සංකල්පය නිරූපණය කරයි. The value of information is at its highest when it is created. With the passage of time, the value of information decreases. This chart illustrates that concept.

දත්ත ජීවන චක්‍ර Data Life Cycles

දත්ත එහි පැවැත්ම පුරාම විවිධ අවස්ථා හරහා ගමන් කරයි. මෙම කොටසෙහි ප්‍රධාන ජීවන චක්‍ර දෙකක් විස්තර කෙරේ: දත්තවල සමස්ත පැවැත්ම (නිර්මාණය සිට ඉවත් කිරීම දක්වා) සහ දත්ත තොරතුරු බවට පත් කිරීමේ නිශ්චිත සැකසුම් චක්‍රය. එක් එක් පියවර ගැන වැඩිදුර දැන ගැනීමට ක්ලික් කරන්න. Data moves through various stages during its existence. This section details two key life cycles: the overall existence of data (from creation to removal) and the specific processing cycle to turn data into information. Click on each step to learn more.

දත්ත ජීවන චක්‍රය (සමස්ත) Data Life Cycle (Overall)

විවිධ ස්ථාන වල විසිරි ඇති දත්ත එකතු කිරීම.Collecting data scattered in different places.

  • සම්මුඛ සාකච්ඡාInterviews
  • නිරික්ෂණයObservation
  • ප්‍රශ්නාවලිQuestionnaires
  • සමීක්ෂණSurveys
  • **පිළියෙල කිරීම:** දත්ත සැකසීමට සුදුසු ලෙස සකස් කිරීම.**Preparation:** Preparing data for processing.
  • **දත්ත ආදානය:** දත්ත පරිගණකයට ඇතුළත් කිරීම.**Data Input:** Entering data into the computer.
  • **දත්ත සැකසීම:** මෘදුකාංග භාවිතයෙන් සකස් කිරීම.**Data Processing:** Processing using software.
  • **ප්‍රතිදානය:** තොරතුරු පරිශීලකයාට යොමු කිරීම.**Output:** Forwarding information to the user.

කල් ඉකුත් වු හෝ නොගැලපෙන දත්ත පද්ධතියෙන් ඉවත් කිරීම.Removing outdated or irrelevant data from the system.

දත්ත සැකසුම් ජීවන චක්‍රය Data Processing Life Cycle

දත්ත රැස් කිරීමේ ක්‍රම (හස්තමය, ස්වයංක්‍රීය).Methods of data collection (Manual, Automatic).

ආදානය කරන දත්ත ගැලපෙනවාදැයි පරීක්ෂා කිරීම.Checking if the input data is suitable.

  • **Validation:** වර්ග පරීක්ෂාව (Type Check), පරාස පරීක්ෂාව (Range Check).**Validation:** Type Check, Range Check.
  • **Verification:** දත්ත නිවැරදිව ඇතුළත් කර ඇත්දැයි බැලීම.**Verification:** Checking if data was entered correctly.
  • **කාණ්ඩ (Batch):** දත්ත එකතු කර පසුව සකස් කිරීම (උදා: වැටුප්).**Batch:** Data is collected and processed later (e.g., payroll).
  • **තත්කාලීන (Real-time):** දත්ත ආදානය කළ සැණින් සකස් කිරීම (උදා: ATM).**Real-time:** Processed immediately upon input (e.g., ATM).

සකස් කළ තොරතුරු පරිශීලකයාට ලබා දීම සහ අනාගත භාවිතය සඳහා ගබඩා කිරීම.Providing processed information to the user and storing it for future use.

මූලික තාක්ෂණයන් Core Technologies

නූතන දත්ත කළමනාකරණය සහ පරිගණක භාවිතය පදනම් වී ඇත්තේ මූලික තාක්ෂණයන් දෙකක් මත ය: වළාකුළු පරිගණක (Cloud Computing) සහ පරිගණක පද්ධතියේ මූලික සංරචක. මෙම කොටස එම සංකල්ප පැහැදිලි කරයි. Modern data management and computing rely on key underlying technologies. This section explains the concepts of Cloud Computing and the basic components that make up a computer system.

වළාකුළු පරිගණක සංකල්පය Cloud Computing Concept

ඕනෑම ස්ථානයක සිට ඕනෑම වේලාවකදී අන්තර්ජාලය හරහා දත්ත වෙත ප්‍රවේශ වීමේ සහ තැන්පත් කිරීමේ හැකියාව.The ability to access and store data from anywhere at any time via the internet.

යටිතල පහසුකම් සේවාවක් ලෙස (IaaS)Infrastructure as a Service (IaaS)

දත්ත ගබඩා කිරීමට අවශ්‍ය අවකාශය සහ සේවාදායක වැනි මූලික සම්පත් ලබා දීම.Provides basic resources like servers and space to store data.

සංවර්ධන පරිසර සේවාවක් ලෙස (PaaS)Platform as a Service (PaaS)

මෘදුකාංග සංවර්ධනයට සහ ධාවනයට අවශ්‍ය පරිසරය (උදා: මෙහෙයුම් පද්ධති) ලබා දීම.Provides the environment (e.g., OS, database) for software development.

මෘදුකාංග සේවාවක් ලෙස (SaaS)Software as a Service (SaaS)

දෘඩාංග තුළ (ROM) ස්ථිරව ඇතුළත් කර ඇති මෘදුකාංග.Software written permanently to a hardware device (ROM).
