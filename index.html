ICT සටහන්

1. Introduction to the Computer (පරිගණකය හැඳින්වීම)

This section covers the fundamental definition of a computer, its historical evolution through different generations, and how computers are classified based on their size, technology, and purpose. මෙම කොටස පරිගණකය පිළිබඳ මූලික අර්ථ දැක්වීම, විවිධ පරම්පරා හරහා එහි ඓතිහාසික පරිණාමය, සහ පරිගණක ඒවායේ ප්‍රමාණය, තාක්ෂණය, සහ අරමුණ අනුව වර්ගීකරණය කරන ආකාරය ආවරණය කරයි.

Definition (හැඳින්වීම)

Sinhala: පරිගණකය යනු මිනිසාගේ ගණිතමය මෙන්ම එදිනෙදා අවශ්‍යතාවයන් සඳහා සැකසු විද්‍යුත් යාන්ත්‍රික උපකරණයකි.

English: A computer is an electromechanical device designed for human mathematical and everyday needs.

2. Computer System Components (පරිගණක පද්ධතියක මුලික සංරචක)

A computer system is not just one item, but a collection of four essential components that work together: Hardware, Software, Liveware, and Firmware. This section defines each one. පරිගණක පද්ධතියක් යනු එක් දෙයක් නොව, එකට ක්‍රියා කරන අත්‍යාවශ්‍ය සංරචක හතරක එකතුවකි: දෘඩාංග, මෘදුකාංග, ජීවාංග, සහ ස්ථිරාංග. මෙම කොටස ඒ සෑම එකක්ම අර්ථ දක්වයි.

1. Hardware (දෘඩාංග)

Sinhala: පරිගණකයට සම්බන්ධ කර ඇති භෞතිකව ස්පර්ශ කළ හැකි සියළුම කොටස්.

English: All the physical parts connected to the computer that can be touched.

2. Software (මෘදුකාංග)

Sinhala: පරිගණකය තුළින් යම් කිසි කාර්යයක් ඉටු කර ගැනීමට දෙනු ලබන උපදෙස් සමූහයක එකතුවකි.

English: Software is a set of instructions given to the computer to perform some activity.

3. Liveware (ජීවාංග)

Sinhala: පරිගණකය හා සම්බන්ඳව ක්‍රියාකරන සියළුම මානව පුද්ගලයින්.

English: All human personnel who interact with the computer. (e.g., Programmer, System Analyst).

4. Firmware (ස්ථිරාංග)

Sinhala: දෘඩාංග තුළ ස්ථිරව ඇතුළත් කර ඇති මෘදුකාංග (උදා: ROM හි ඇති BIOS).

English: Software permanently embedded in hardware (e.g., BIOS in ROM).

3. Computer Hardware In-Depth (දෘඩාංග පිළිබඳ ගැඹුරින්)

This section dives deeper into the core hardware components. Use the tabs below to explore the CPU (the 'brain'), the different types of computer memory, and the motherboard that connects everything. මෙම කොටස මූලික දෘඩාංග සංරචක ගැඹුරින් ගවේෂණය කරයි. 'මොළය' (CPU), විවිධ පරිගණක මතක වර්ග, සහ සියල්ල සම්බන්ධ කරන මව් පුවරුව ගවේෂණය කිරීමට පහත ටැබ් භාවිතා කරන්න.

Central Processing Unit (CPU) (මධ්‍ය සැකසුම් ඒකකය)

English: The primary component of a computer system. Also considered the heart/brain of the computer. It follows the Fetch-Execute Cycle to run instructions.

Sinhala: පරිගණක පද්ධතියක මූලිකම අංගය. පරිගණකයේ හදවත / මොළය ලෙස ද සලකයි. එය උපදෙස් ක්‍රියාත්මක කිරීමට සෙවුම්-ඉෂ්ට චක්‍රය අනුගමනය කරයි.

ALU (තාර්කික ඒකකය)

Performs all mathematical and logical operations.

Control Unit (CU) (පාලන ඒකකය)

Directs the entire operation of the computer.

Registers (රෙජිස්තර්)

Small, high-speed storage locations within the CPU.

Memory (මතකය)

Memory is organized in a hierarchy based on speed, cost, and size. Faster memory is smaller and more expensive.

මතකය වේගය, පිරිවැය, සහ ප්‍රමාණය මත පදනම්ව ධුරාවලියක සංවිධානය කර ඇත. වේගවත් මතකය කුඩා වන අතර මිල අධික වේ.

Read-Only Memory (ROM)

Non-volatile (ස්ථිර) memory used to store firmware like BIOS. Types include PROM, EPROM, EEPROM.

Random Access Memory (RAM)

Volatile (අස්ථිර) memory that stores data temporarily while the computer is running. Types include DRAM and SRAM.

Motherboard (මව් පුවරුව)

The main circuit board that connects all hardware components together.

සියලුම දෘඩාංග සංරචක එකට සම්බන්ධ කරන ප්‍රධාන පරිපථ පුවරුව.

Bus (බස්)

A set of wires used to move data between components (Data bus, Address bus, Control bus).

Chipset (චිප් සෙට්)

Small circuits that control the operation and data flow to and from the CPU.

4. Operating Systems (මෙහෙයුම් පද්ධති)

The Operating System (OS) is the most important software on your computer. It manages all hardware and software resources and provides an interface for you to interact with the machine. Explore its key functions below. මෙහෙයුම් පද්ධතිය (OS) යනු ඔබේ පරිගණකයේ ඇති වැදගත්ම මෘදුකාංගයයි. එය සියලුම දෘඩාංග සහ මෘදුකාංග සම්පත් කළමනාකරණය කරන අතර යන්ත්‍රය සමඟ අන්තර් ක්‍රියා කිරීමට ඔබට අතුරු මුහුණතක් සපයයි. පහතින් එහි ප්‍රධාන කාර්යයන් ගවේෂණය කරන්න.

Main Functions (ප්‍රධාන කාර්යයන්)

1. Process Management (ක්‍රියායන කළමනා කරණය)

Manages the execution of programs (processes).

2. Resource Management (සම්පත් කළමනාකරණය)

Allocates CPU time, memory, and devices.

3. User Interface (UI) (පරිශීලක අතුරු මුහුණත්)

Provides a way for the user to interact (CLI or GUI).

4. Security & Protection (ආරක්ෂාව)

Protects the system from threats and unauthorized access.

File Management (ගොනු කළමනාකරණය)

The OS controls how data is stored and retrieved using file systems.

ගොනු පද්ධති භාවිතයෙන් දත්ත ගබඩා කරන සහ ලබා ගන්නා ආකාරය OS මගින් පාලනය කරයි.

Process Management (ක්‍රියායන කළමනාකරණය)

The OS manages all running programs (processes), including their states and scheduling.

Scheduling Policies (නියමකරණ ප්‍රතිපත්ති)

  • First-Come, First-Serve (FCFS): Processes run in the order they arrive.
  • Shortest Job First (SJF): The process with the shortest execution time runs next.
  • Priority (ප්‍රමුඛතා): Processes are assigned a priority, and higher priority processes run first.
  • Round Robin: Each process gets a small time slice (quantum) to run before moving to the back of the queue.

Memory Management (මතකය කළමනාකරණය)

The OS manages the computer's main memory, most importantly by using Virtual Memory.

පරිගණකයේ ප්‍රධාන මතකය OS මගින් කළමනාකරණය කරයි, විශේෂයෙන් අතථ්‍ය මතකය භාවිතා කරයි.

Virtual Memory (අතථ්‍ය මතකය)

English: A technique that uses secondary storage (hard disk) as if it were part of main memory (RAM). This allows the computer to run larger programs than it could fit in RAM alone.

Sinhala: ද්විතීයික ආචයනය (දෘඪ තැටිය) ප්‍රධාන මතකයේ (RAM) කොටසක් ලෙස භාවිතා කරන තාක්‍ෂණයකි. RAM එකට පමණක් සරිලනවාට වඩා විශාල වැඩසටහන් ක්‍රියාත්මක කිරීමට මෙය පරිගණකයට ඉඩ සලසයි.

Paging (පිටුකරණය): Virtual memory is divided into "Pages," and physical memory is divided into "Frames."

Address Translation (ලිපින පරිවර්තනය): The CPU's "Virtual Address" is translated into a "Physical Address" (in RAM) by the Memory Management Unit (MMU).

5. Data and Information (දත්ත සහ තොරතුරු)

This section defines the fundamental concepts of data, information, and the process (Data Life Cycle) that transforms one into the other. It also covers how we ensure data is correct. මෙම කොටස දත්ත, තොරතුරු, සහ එකක් අනෙකක් බවට පරිවර්තනය කරන ක්‍රියාවලිය (දත්ත ජීවන චක්‍රය) පිළිබඳ මූලික සංකල්ප අර්ථ දක්වයි. දත්ත නිවැරදි බව අපි සහතික කරන්නේ කෙසේද යන්න ද එය ආවරණය කරයි.

Data (දත්ත)

English: Numbers, words, symbols, images that cannot be drawn to definite conclusions.

Sinhala: නිශ්චිත නිගමන වලට එළඹිය නොහැකි අංක, වචන, සලකුණු, රූප.

Information (තොරතුරු)

English: Data prepared in such a way that specific conclusions or decisions can be reached.

Sinhala: නිශ්චිත නිගමනයකට හෝ තීන්දු වලට එළඹිය හැකි ලෙස සකස් කරන ලද දත්ත.

Data Correctness (දත්ත නිවැරදි බව)

Validation (වලංගුතාව)

English: Checks if data is reasonable and in the correct format (e.g., Range Check, Type Check).

Verification (සත්‍යාපනය)

English: Checks if the data entered matches the original source (e.g., double entry).

6. Data Communication & Networking (දත්ත සන්නිවේදනය)

This section covers how computers talk to each other, from the basic concepts and physical media to the complex models and security protocols that make the internet work. පරිගණක එකිනෙකා සමඟ සන්නිවේදනය කරන ආකාරය, මූලික සංකල්ප සහ භෞතික මාධ්‍යවල සිට අන්තර්ජාලය ක්‍රියාත්මක කරන සංකීර්ණ ආකෘති සහ ආරක්ෂක ප්‍රොටෝකෝල දක්වා මෙම කොටසින් ආවරණය වේ.

Communication Basics (සන්නිවේදන මූලිකාංග)

Transmission Modes (ආකාර)

  • Simplex (ඒකපථ): One way only (e.g., Radio).
  • Half-duplex (අර්ධ): Two ways, but not at the same time.
  • Full-duplex (පුර්ණ): Two ways simultaneously.

Transmission Media (මාධ්‍ය)

  • Guided (නියමු): Twisted Pair, Co-axial, Fiber Optics.
  • Unguided (නියමු නොවන): Radio waves, Microwaves.

Bus (බස්)

Single main cable.

Addressing (ලිපිනගත කිරීම)

MAC Address

English: A unique 48-bit hardware address burnt into the NIC.

Sinhala: NIC (ජාල අතුරුමුහුණත් කාඩ්පත) මත ඇති අනන්‍ය 48-bit දෘඩාංග ලිපිනය.

IP Address (IP ලිපිනය)

English: A logical address used to identify devices on a network. IPv4 is 32-bit, while the newer IPv6 is 128-bit.

Sinhala: ජාලයක උපාංග හඳුනා ගැනීමට භාවිතා කරන තාර්කික ලිපිනය. IPv4 32-bit වන අතර, නව IPv6 128-bit වේ.

Router (මංහසුරුව)

Bridge (සේතු)

Connects two similar LANs.

Repeater (පුර්න යෝජකය)

7. ICT in Society (සමාජය කෙරෙහි ICT)

This final section explores the modern application of ICT, focusing on Cloud Computing and the broad impacts (both good and bad) that technology has on our daily lives. මෙම අවසාන කොටස ICT හි නවීන යෙදුම්, Cloud Computing කෙරෙහි අවධානය යොමු කරමින්, සහ තාක්ෂණය අපගේ එදිනෙදා ජීවිතයට ඇති කරන පුළුල් බලපෑම් (හොඳ සහ නරක) ගවේෂණය කරයි.

Cloud Computing (වළාකුළු පරිගණක සංකල්පය)

English: The ability to access and store data and run applications over the internet from anywhere in the world at any time.

Sinhala: ලොව ඕනෑම ස්ථානයක සිට ඕනෑම වේලාවකදී අන්තර්ජාලය හරහා දත්ත වෙත ප්‍රවේශ වීමේ, තැන්පත් කිරීමේ සහ යෙදුම් ක්‍රියාත්මක කිරීමේ හැකියාව.

SaaS (Software as a Service)

Using software over the internet (e.g., Gmail).

PaaS (Platform as a Service)

A platform for developing applications.

IaaS (Infrastructure as a Service)

Renting virtualized hardware (servers, storage).

Impact of ICT (ICT හි බලපෑම)

Beneficial (හිතකර)

  • ATMs, E-banking
  • Advanced Communication
  • Modern Medicine & Surgery
  • Education (E-learning)

Negative (අහිතකර)

  • Rising unemployment (due to automation)
  • Privacy problems
  • Addiction
  • Health issues (e.g., eye strain)
